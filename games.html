Games

Contained here are all of the games I have built or that I'm working on. I started developing games as a fun way to build on my existing programming knowledge and to improve my proficiency in JavaScript and with the handling and organisation of larger-scale projects. Feel free to have a browse and try out some of them!

Unfortunately, due to the restrictions of this site's hosting, I am unable to run any server-side applications. Alas, all of these games have been written using JavaScript. However I am currently working on setting up a static-server for the site so that some of these games can support more than just locally-based multiplayer.

Breakout Clone

A replica of Atari's 1976 classic 'Breakout'

Creating this game was my introductory exercise using JavaScript and HTML Canvas. Set the paddle width and ball speed to adjust the game's difficulty. Gain a life by passing a level without dying!
Controls:

  • Left & right arrow keys - adjust paddle position
  • Esc - pause game
  • 's' - display settings

Mini Croquet

Arcade version of the popular British lawn sport

A considerable step up from the breakout clone, building this game involved more complicated gaming logic and physics. Navigate the course by hitting the ball through the correct gate and finsh by striking the post in the center!
Controls:

  • Cursor - control direction of play and roqueting position
  • Spacebar - use to adjust the power

Pool

Miniature version of a personal favourite

Building on some of the functionality developed in 'Mini Croquet', I decided to create a miniature pool game. This game is very much in its infancy. It requires a more sophisticated method of solving momentum problems to deal with the increased number of balls. I would like to develop the game such that multiple varieties of the sport can be played and with an added 'speed mode'.
Controls:

  • Cursor - control direction of play
  • Spacebar - use to adjust the power
