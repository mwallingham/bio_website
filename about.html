About me

My name is Mark Allingham, I am 23 years old and from Northern Ireland. Whilst studying engineering at university, I developed a keen insterest in coding and a variety of programming languages. During lockdown I wanted to use any spare time I had developing new and existing skills. So, I started this website as a project to help me expand my programming knowledge and to introduce me to new programming languages. I keep this site active to act as a source of motivation for working on technical projects which in turn allows me to continue learning and developing as an engineer. For the most part, the code used in this website was written by me - with the exception of some of the CSS content used to set the theme, which was sourced from Bootstrap.
